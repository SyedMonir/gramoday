## How to run this project

Clone the project

```bash
  git clone (https://github.com/SyedMonir/gramoday.git)
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
